# KnowBrain（知脑）— 企业私有知识大脑

> 把公司文档投喂进去，员工用自然语言提问，AI 精准回答 + 溯源原文。
> 纯私有化部署，数据不出企业内网。

---

## 解决什么问题

| 场景 | 没有 KnowBrain | 有了 KnowBrain |
|------|------|------|
| 新员工问「VPN 怎么配」 | 追着老员工问，等半天回复 | 输入问题 → 3 秒出答案，附操作步骤 |
| 查「年假几天、婚假怎么请」 | 翻 HR 制度 PDF，看半天找不到 | 自然语言问 → 直接回答 + 引用原文段落 |
| 电脑中病毒了怎么办 | 百度搜 → 信息不对口 / 过时 | 基于公司安全预案 → 精准应急步骤 |
| IT 运维每天答重复问题 | 50% 时间在做客服 | AI 自动解答，人工只处理复杂问题 |
| 员工离职，经验流失 | 人走了知识也走了 | 文档永久沉淀，AI 持续可查 |

## 怎么用（三步）

```
① 上传文档     ② 员工提问       ③ AI 回答 + 溯源
  PDF/Word     「年假几天？」    → 5天年假，工作满1年后享受
  Excel/PPT     「VPN怎么连？」   → 打开客户端→输入地址...
   Markdown                         [来源: 员工手册 V3.2]
```

## 跟 ChatGPT 的区别

| | ChatGPT / Kimi | KnowBrain |
|------|:--:|:--:|
| 数据安全 | 数据上传到公网服务器 | **数据留在企业内网** |
| 回答来源 | 通用互联网知识 | **基于你的公司文档** |
| 可验证 | 不知道答案从哪来的 | **每条回答标注来源文档** |
| 权限管控 | 没有 | **按部门/角色控制文档可见范围** |
| 部署方式 | SaaS，必须联网 | **Docker 一键部署，内网运行** |

> KnowBrain 不是替代 ChatGPT，是解决「公司内部知识」这个 ChatGPT 覆盖不了的盲区。

---

## 技术能力

- **混合检索**：语义理解 + 关键词匹配，既懂「电脑卡」也懂「系统性能下降」
- **智能改写**：你问「咋弄」「破电脑」，系统自动转为正式术语检索
- **多格式解析**：PDF、Word、Excel、PPT、Markdown 直接上传
- **接入企微/钉钉/飞书**：在 IM 里直接 @机器人 提问
- **评测体系**：内置 RAG 质量评测，持续追踪检索效果

## 部署（15 分钟）

```bash
# 前置：Docker + 8GB 内存 + 任意 Linux/Windows/Mac
git clone https://github.com/xbchenf/KnowBrain
cd KnowBrain
cp .env.example .env   # 填入 LLM API Key
docker compose up -d    # 一键启动

# 打开浏览器 → http://localhost → 上传文档 → 开始提问
```

最低配置：4 核 CPU、16GB 内存、50GB 磁盘（不含 GPU）
