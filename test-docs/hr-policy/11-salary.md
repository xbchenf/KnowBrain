# 薪酬管理制度（工资发放与个税扣除）

> 文件编号：HR-2024-011  
> 版本号：V3.2  
> 生效日期：2024年1月1日  
> 审批人：人力资源总监  
> 适用范围：某科技有限公司全体正式员工

---

## 概述

为规范某科技有限公司（以下简称"公司"）薪酬管理工作，保障员工合法权益，依据《中华人民共和国劳动法》《中华人民共和国个人所得税法》及公司相关规章制度，制定本制度。本制度适用于公司全体正式员工，实习员工及劳务派遣人员参照相关条款执行。

薪酬管理遵循"公平、透明、及时、合规"四项基本原则，由人力资源部（以下简称HR）统一归口管理，财务部协同实施。员工如对工资有任何疑问，可联系HR薪酬专员（分机6667）。

---

## 薪资规定

### 发薪日

公司实行月薪制，每月发薪日为**10日**。如发薪日恰逢法定节假日或周末，则**提前至最近一个工作日**发放。工资通过**银行代发**方式直接转入员工本人在公司备案的工资卡账户，不发放现金。

新入职员工于入职当月开始计薪，按**实际工作天数**折算首月工资。计算方式为：月基本工资 ÷ 当月应出勤天数 × 实际工作天数。

如发薪日未收到工资到账通知，请先确认银行卡状态是否正常，如确认异常请于发薪日次日12:00前联系HR薪酬专员核查。

### 工资构成

员工月度工资由以下部分组成：

| 项目 | 说明 |
|------|------|
| 基本工资 | 劳动合同约定岗位工资 |
| 绩效工资 | 根据月度绩效考核结果浮动 |
| 加班费 | 依法计算工作日/休息日/法定节假日加班费 |
| 各类补贴 | 餐补、交通补贴、通讯补贴、高温补贴等 |
| 扣款项 | 社保个人部分、住房公积金个人部分、个人所得税、迟到早退扣款等 |

### 工资条

员工可通过以下路径查看**工资条**：

1. 登录公司OA系统
2. 进入「薪酬中心」模块
3. 点击「我的工资」
4. 选择对应月份查看当月及历史工资明细

工资条内容包含基本工资、绩效工资、加班费、各类补贴、社保代扣、个税代扣等详细信息。每月工资条于发薪日同步发布。如对工资条内容有疑问，请在发薪后5个工作日内向HR薪酬专员反馈，逾期视为确认无误。

## 个人所得税管理

公司作为法定扣缴义务人，每月按照**累计预扣法**为员工代扣代缴个人所得税。

### 专项附加扣除

员工可享受以下专项附加扣除项目（如符合条件）：

- 子女教育
- 继续教育
- 大病医疗
- 住房贷款利息
- 住房租金
- 赡养老人
- 3岁以下婴幼儿照护

符合条件的员工请在**个人所得税App**自行申报专项附加扣除信息，公司每月从税务系统同步扣除额度并在计算个税时予以抵扣。建议员工于每年12月在个税App中确认下一年度的扣除信息，以免影响次年扣除。

### 年度汇算清缴

每年**3月1日至6月30日**为上一年度综合所得年度汇算清缴期。公司HR将在每年3月初发送全员通知邮件，提醒员工及时办理汇算。

如对个人所得税计算或扣除政策有疑问，请联系HR薪酬专员（分机6667），或拨打税务服务热线12366咨询。

---

## 注意事项

1. 员工银行卡信息如有变更，须提前5个工作日通过OA系统更新，否则由此造成的工资发放延误由员工自行承担。
2. 因员工个人原因（如未及时提交考勤异常说明、未确认绩效评分等）导致的薪酬核算争议，HR将根据现有材料核算，后续不予补发调整。
3. 离职员工的最后薪资于离职手续办理完毕后，随最近一个发薪日发放。
4. 本制度中未明确之特殊情形，由HR报公司管理层审批后个案处理。
