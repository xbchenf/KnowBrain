# 新员工 IT 入职指南

> 欢迎加入 KnowBrain！本文档帮助你在一小时内完成所有 IT 环境配置。如有疑问联系 IT 服务台（分机 8888）。

---

## 一、入职前准备（HR 通知 IT 后自动完成）

在你入职前一天，IT 已自动完成以下准备：

- [x] 域账号创建（员工号 + 初始密码）
- [x] 企业邮箱开通（`姓名拼音@knowbrain.com`）
- [x] 企业微信账号开通
- [x] OA 系统账号创建
- [x] 办公电脑分配 + 系统预装
- [x] 各系统权限初始化（按岗位模板）

## 二、第一天配置清单（按顺序完成，约 45 分钟）

### 2.1 登录办公电脑（5 分钟）

1. 按下主机电源键，等待 Windows 启动
2. 登录界面输入：
   - 用户名：`knowbrain\KB00XXX`（员工号见 HR 发的入职邮件）
   - 密码：HR 邮件中的初始密码
3. 首次登录后系统会强制修改密码，设置你的新密码：
   - 长度 ≥ 10 位
   - 至少包含大写字母、小写字母、数字、特殊字符中的 3 类

### 2.2 配置企业微信（5 分钟）

1. 手机下载「企业微信」App（iOS App Store / Android 应用商店）
2. 选择「手机号登录」，使用入职时登记的手机号
3. 登录后确认能看到「KnowBrain」企业
4. 在工作台中找到「IT 服务台」应用
5. 电脑端可同步安装企业微信桌面版

### 2.3 配置企业邮箱（10 分钟）

详细步骤参考 [企业邮箱配置教程](#)，快速版：

1. 打开 https://mail.knowbrain.com 使用域账号登录
2. 修改邮箱初始密码（必须修改，初始密码同域账号）
3. 绑定手机号和备用邮箱（用于找回密码）
4. 设置邮件签名（模板见企业邮箱配置教程）
5. 安装 Outlook 客户端：
   - 打开开始菜单 → Microsoft Outlook
   - 输入邮箱地址 → 自动完成 Exchange 配置

### 2.4 连接 WiFi / 有线网络

办公室已覆盖 WiFi，SSID：`KB-Office`，域控电脑自动连接。

如需使用个人设备连接 WiFi，详见 [WiFi 使用指南](#)。

### 2.5 安装常用软件（15 分钟）

打开「开始菜单 → Software Center（软件中心）」安装以下软件：

**必须安装：**

| 软件 | 用途 | 安装方式 |
|------|------|---------|
| Microsoft Office 365 | 文档办公 | Software Center 一键安装 |
| Microsoft Teams | 视频会议 | Software Center 一键安装 |
| CrowdStrike Falcon | 安全防护 | IT 已预装 |
| OneDrive for Business | 云盘同步 | Software Center 一键安装 |

**按需安装（根据岗位选择）：**

| 软件 | 适用岗位 | 用途 |
|------|---------|------|
| VS Code | 开发 | 代码编辑器 |
| IntelliJ IDEA | Java 开发 | 后端 IDE |
| DBeaver | 开发/数据分析 | 数据库管理 |
| Figma | 产品/设计 | UI 设计 |
| Postman | 开发/测试 | API 调试 |
| Navicat | 开发 | 数据库管理（需申请 License） |
| WPS Office | 通用 | 国产办公套件（可选替代 Office） |

### 2.6 配置 VPN（如需远程办公）

1. 登录 OA → IT 服务 → VPN 权限申请（入职当天可提交）
2. IT 审批后下载客户端安装
3. 详细配置见 [VPN 配置手册](#)

请将以下系统加入浏览器书签栏：

## 三、权限申请指南

入职时 IT 已按你的岗位模板预配基础权限。如需额外权限，按以下方式申请：

| 权限类型 | 申请方式 | 审批人 | 生效时间 |
|---------|---------|--------|---------|
| 代码仓库（GitLab） | OA → IT 服务 → 代码仓库权限 | 技术主管 | 2 小时内 |
| 生产服务器 SSH | OA → IT 服务 → 服务器访问权限 | CTO + IT | 24 小时内 |
| 财务系统 | OA → IT 服务 → 业务系统权限 | 财务负责人 | 4 小时内 |
| 数据库只读 | OA → IT 服务 → 数据库权限 | 技术主管 + DBA | 4 小时内 |
| 数据库读写 | OA → IT 服务 → 数据库权限 | 技术主管 + DBA + IT | 48 小时内 |
| 外部网络访问（白名单） | OA → IT 服务 → 外网访问申请 | 部门负责人 | 4 小时内 |

## 五、第一天配置检查清单

- [ ] 能登录办公电脑（新密码已设）
- [ ] 企业微信已登录且能收到消息
- [ ] 企业邮箱已登录（网页 + Outlook 客户端）
- [ ] 能连接 WiFi / 有线网络正常
- [ ] 办公必备软件已安装
- [ ] 浏览器已收藏 OA 和常用系统
- [ ] VPN 已申请（如需）
- [ ] 打印机测试页已打印（确认驱动正常）
- [ ] OneDrive 已登录且文件夹同步正常

> 💡 **提示**：第一天上午 IT 服务台会有专人巡楼，可现场协助配置。下午有任何问题随时企业微信找「IT 服务台」。
