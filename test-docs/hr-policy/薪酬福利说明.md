## 一、薪酬体系

### 1.1 薪酬结构

公司实行「固定工资 + 浮动工资 + 长期激励」的全面薪酬体系：

| 薪酬要素 | 占比 | 发放周期 | 说明 |
|---------|------|---------|------|
| 基本工资 | 月薪的 70% | 月度 | 固定发放 |
| 绩效工资 | 月薪的 30% | 月度 | 按绩效考核系数浮动 |
| 年终奖金 | 1-6 倍月薪 | 年度 | 公司业绩 × 个人绩效 |
| 项目奖金 | 按项目 | 里程碑节点 | 研发/交付岗位 |
| 期权激励 | — | 4 年归属 | 核心员工 |

### 1.2 薪酬等级

各职级对应薪酬带宽（以下为税前月薪范围，单位：元）：

| 职级 | 月薪范围 | 中位值 |
|------|---------|--------|
| P1 初级 | 8,000 — 12,000 | 10,000 |
| P2 初级+ | 10,000 — 18,000 | 14,000 |
| P3 中级 | 16,000 — 25,000 | 20,000 |
| P4 中级+ | 22,000 — 35,000 | 28,000 |
| P5 高级 | 30,000 — 45,000 | 38,000 |
| P6 高级+ | 40,000 — 60,000 | 50,000 |
| P7 资深 | 55,000 — 80,000 | 65,000 |
| P8 专家/架构师 | 70,000 — 100,000 | 85,000 |

> 实际薪酬根据候选人经验、能力、市场水平综合评定，不限于上述范围。

## 二、发薪管理

### 2.1 发薪时间

| 项目 | 时间 |
|------|------|
| 工资发放日 | 每月 10 日（遇节假日提前至最近工作日） |
| 发薪方式 | 银行代发至招商银行工资卡 |
| 工资条发送 | 每月 10 日企业微信推送电子工资条 |

### 2.2 工资计算规则

| 场景 | 计算方式 |
|------|---------|
| 满勤 | 基本工资 + 绩效工资 × 绩效系数 |
| 缺勤 | 日薪 = 月基本工资 / 21.75 天 |
| 入职当月 | 日薪 × 实际工作日天数 |
| 离职当月 | 日薪 × 截止离职日的实际工作日天数 |
| 事假扣除 | 日薪 × 事假天数（扣除基本工资部分） |
| 病假扣除 | 1-2 天不扣；3 天以上扣日薪 × 20% |

### 2.3 加班费计算

| 加班类型 | 计算基数 | 倍数 |
|---------|---------|------|
| 工作日加班 | 月基本工资 / 21.75 / 8 | 1.5 倍 |
| 休息日加班 | 月基本工资 / 21.75 / 8 | 2 倍 |
| 法定节假日 | 月基本工资 / 21.75 / 8 | 3 倍 |

> 加班费按小时计算，最小计算单位为 0.5 小时。

## 三、奖金制度

### 3.1 年终奖金

**计算方式：** 年终奖 = 月薪 × 公司系数 × 个人绩效系数

| 个人绩效等级 | 个人系数 |
|------------|---------|
| S（卓越） | 3.0 — 6.0× |
| A（优秀） | 2.0 — 3.0× |
| B+（良好） | 1.5 — 2.0× |
| B（合格） | 1.0 — 1.5× |
| C（待改进） | 0 — 0.5× |
| D（不合格） | 0 |

**公司系数**根据年度营收目标达成率确定：
- 达成 ≥ 120%：公司系数 1.5
- 达成 100%-120%：公司系数 1.2
- 达成 80%-100%：公司系数 1.0
- 达成 < 80%：公司系数 0.6

**发放条件**：
- 年终奖发放日（次年 3 月 31 日）仍在职
- 入职不满 6 个月按实际月份折算（入职满 6 个月全额计算）

### 3.2 项目奖金

适用于研发和项目交付岗位：

| 类别 | 说明 |
|------|------|
| 发放节点 | 项目关键里程碑完成后 1 个月内 |
| 奖金池 | 项目预算的 5%-10% |
| 分配依据 | 项目角色重要性 × 个人贡献度 × 工作时长占比 |
| 审批流程 | 项目经理提案 → 部门负责人审核 → HR/财务审核 |

### 3.3 即时奖励

用于激励突出贡献和紧急攻关：

- **金额范围**：500 — 5,000 元
- **场景**：紧急故障修复、技术难题攻克、客户高度好评、合理化建议被采纳
- **提名方式**：任何人可提名，部门负责人审批后即时发放

- 公积金贷款额度：北京最高 120 万（单人）/ 200 万（夫妻）
