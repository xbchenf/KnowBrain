# 费用报销管理制度

> 版本：V3.0 | 生效日期：2026-01-01 | 发布部门：财务部

---

## 一、总则

本制度规范 KnowBrain 公司全体员工因公产生费用的报销流程、标准和审批权限。员工报销须遵循「真实合理、手续完备、标准合规」的原则。

---

## 二、报销基本流程

```
员工提交报销申请（OA 系统）
   → 直属上级审批（1-2 个工作日）
   → 财务审核（1-3 个工作日）
   → 财务总监审批（单笔 ≥ 5,000 元）
   → 出纳打款（审批通过后次周五统一打款）
```

| 环节 | 时限 | 说明 |
|------|------|------|
| 提交时限 | 费用发生后 30 天内 | 逾期不予报销 |
| 上级审批 | 1-2 个工作日 | 出差报销需填写出差申请单 |
| 财务审核 | 1-3 个工作日 | 发票不合规将退回重提 |
| 打款时间 | 审批通过后次周五 | 统一银行代发至工资卡 |

## 三、发票要求

### 3.1 发票类型

| 费用类别 | 要求 |
|---------|------|
| 所有报销 | 必须开具**公司抬头的增值税专用发票**或**普通发票** |
| 餐饮招待 | 需附带消费小票/水单 |
| 机票 | 航空运输电子客票行程单（或电子发票 + 登机牌） |
| 火车票 | 铁路电子客票（或纸质车票） |
| 住宿 | 增值税专用发票 + 住宿清单（水单） |
| 打车 | 电子发票（滴滴等平台直接在 App 内申请） |

### 3.2 发票信息

| 项目 | 内容 |
|------|------|
| 公司全称 | 知脑科技有限公司（KnowBrain） |
| 纳税人识别号 | 9111 0108 MAXX XXXX XXX |
| 开户行及账号 | 招商银行北京分行望京支行 1109 XXXX XXXX XXXX |

### 3.3 发票不合规情形（财务将退回）

- 发票抬头错误或空白
- 发票信息模糊、涂改
- 电子发票重复报销（财务系统自动查重拦截）
- 与实际消费不符的代开发票
- 发票金额、日期与实际业务不匹配
- 增值税专用发票未按要求开具或认证期已过

## 四、差旅费用标准

### 4.1 交通费

| 职级 | 飞机 | 火车 | 市内交通 |
|------|------|------|---------|
| P5 及以上 / M1 及以上 | 经济舱 | 高铁二等座 | 出租车 / 网约车 实报实销 |
| P3-P4 | 经济舱（单程 > 6h 可申请） | 高铁二等座 | 网约车实报实销 |
| P1-P2 | 火车（单程 > 8h 可申请高铁） | 高铁二等座/动车 | 公共交通为主 |

> 📌 机票统一通过公司指定差旅平台（携程商旅）预订，无需个人垫付。

### 4.2 住宿费（标准/晚，含税）

| 城市等级 | P5+/M1+ | P3-P4 | P1-P2 |
|---------|---------|-------|-------|
| 一线（北上广深） | 600 元 | 450 元 | 350 元 |
| 二线（省会/计划单列市） | 450 元 | 350 元 | 250 元 |
| 三四线及其他 | 350 元 | 250 元 | 200 元 |

> 超出标准部分自理。同性双人出差建议合住标间，住宿标准可上浮 30%。

### 4.3 出差补贴

| 类型 | 标准 | 说明 |
|------|------|------|
| 餐补 | 80 元/天 | 无需发票，按出差天数包干发放 |
| 市内交通补 | 50 元/天 | 无需发票，按出差天数包干发放 |
| 境外差旅 | 按出差国家/地区另定标准 | 需提前报财务总监审批 |

### 4.4 出差申请

1. OA → 行政服务 → 出差申请
2. 填写：出差事由、目的地、起止日期、预计费用
3. 审批：主管 → 部门负责人（单次 > 5 天或跨省）
4. 出差结束后 10 个工作日内完成费用报销

## 六、交通与通讯

### 6.1 加班打车

- **条件**：加班至 21:00 以后
- **范围**：公司 ↔ 居住地单程
- **上限**：150 元/次，实报实销
- **工具**：滴滴企业版（已集成 OA，自动记录行程）

### 6.2 因公外出

- 因公外出拜访/办事交通费实报实销
- 优先使用公共交通，紧急/不便时可选网约车

### 6.3 通讯费

- 正式员工：100 元/月通讯补贴，随工资发放（无需发票）
- 特定岗位（销售/客户经理）：可额外报销业务通话费，凭发票实报实销

## 八、审批权限

| 单笔报销金额 | 审批链 |
|------------|--------|
| < 1,000 元 | 直属上级 → 财务 |
| 1,000 — 4,999 元 | 直属上级 → 部门负责人 → 财务 |
| 5,000 — 19,999 元 | 直属上级 → 部门负责人 → 财务总监 |
| ≥ 20,000 元 | 直属上级 → 部门负责人 → 财务总监 → CFO |

> 各审批节点超 3 个工作日未处理，系统自动提醒并升级。

## 九、常见问题

### Q1：发票丢失怎么办？

- 电子发票：从开票方重新获取 PDF，不可重复报销
- 纸质发票丢失：提供开票方盖章的发票复印件或记账联复印件 + 情况说明（主管签字）
- 住宿/机票发票丢失：联系酒店/航空公司补开

### Q2：多人一起出差如何报销？

- 原则上各人报销各自费用
- 住宿费由一人统一支付的可由该员工统一报销，附同住人员名单
- 聚餐费用由最高职级人员报销

### Q3：个人垫付后报销多久到账？

- 提交报销后 5-8 个工作日内到账
- 大额垫付（> 5,000 元）可申请预支，审批后财务直接支付

### Q4：哪些费用不能报销？

- 个人消费（购物、娱乐、个人用餐）
- 无公司抬头发票的费用
- 超出公司标准的部分
- 罚款（交通违章、逾期罚金等）
- 礼品卡/购物卡（需走礼品采购流程）
- 小费（境外差旅除外）

## 十、OA 报销操作指引

1. 登录 OA → 行政服务 → 费用报销
2. 选择报销类型（差旅/招待/交通/培训/其他）
3. 上传发票照片/PDF（系统自动 OCR 识别发票信息）
4. 填写报销事由（简要说明费用与工作的关联）
5. 确认金额、检查无误后提交
6. 在「我的报销」中追踪审批进度
7. 纸质发票原件按季度统一提交给财务归档（电子发票无需交纸质）
