# 公司无线网络（WiFi）使用与管理规范

> 文档版本：v2.0 | 生效日期：2025-02-01 | 责任部门：IT 基础设施部 | 适用范围：某科技有限公司总部及各地分公司办公区

---

## 概述

某科技有限公司在全部办公区域部署了企业级无线网络覆盖，采用 WPA3-Enterprise 加密协议，配合 RADIUS 服务器实现基于企业账号的统一身份认证。办公区提供两套无线网络：面向正式员工的企业内网 WiFi 和面向外部访客的临时 WiFi，二者网络隔离，访客网络不可访问内网资源。

无线网络设备由 Aruba 无线控制器和分布式 AP 组成，每层办公区部署不少于 4 个 AP 节点，支持 2.4GHz 和 5GHz 双频自动切换，理论覆盖半径 25 米。

## WiFi 网络信息

| 项目 | 企业网络 | 访客网络 |
|------|---------|---------|
| SSID | **Company-WiFi** | **Company-Guest** |
| 加密方式 | WPA3-Enterprise (802.1X) | WPA2-Personal |
| 认证方式 | 企业账号 + 密码 | 手机号码 + 短信验证码 |
| 有效期 | 账号有效期内持续可用 | 8 小时，超时须重新验证 |
| 可访问资源 | 内网 + 互联网 | 仅互联网（禁止访问内网） |
| 带宽限制 | 50 Mbps/设备 | 10 Mbps/设备 |
| 适用对象 | 正式员工、实习生 | 来访客户、供应商、面试候选人 |

## 连接配置步骤（企业网络 Company-WiFi）

### Windows 系统

1. 点击任务栏右下角网络图标，在 WiFi 列表中找到 **Company-WiFi**
2. 点击「连接」，系统弹出凭据输入窗口
3. 用户名输入 `员工编号@company.com`（如 `ZH00123@company.com`），密码输入企业账号密码
4. 首次连接时会弹出证书信任提示，点击「连接」确认
5. 连接成功后，WiFi 图标显示为已连接且无感叹号

### macOS 系统

1. 点击菜单栏 WiFi 图标，选择 **Company-WiFi**
2. 在弹出的 802.1X 认证窗口中输入企业账号和密码
3. 勾选「记住此网络」以便后续自动连接
4. 系统提示证书验证，点击「继续」信任证书

### 手机端（iOS/Android）

1. 打开设置 → WiFi，选择 **Company-WiFi**
2. 身份（Identity）输入企业账号，密码输入企业账号密码
3. Android 设备须在「CA 证书」中选择「不验证」，「EAP 方法」选择「PEAP」，阶段 2 认证选择「MSCHAPV2」

## WiFi 连不上/连接失败的排查指南

### 标准排查流程

| 步骤 | 操作 | 检查要点 |
|------|------|---------|
| 第 1 步 | 确认 WiFi 开关已打开 | 笔记本检查物理开关或 Fn+功能键，手机检查系统设置 |
| 第 2 步 | 确认连接的是正确 SSID | 务必选择 **Company-WiFi**，不要误选 Company-Guest 或其他同名干扰信号 |
| 第 3 步 | 忘记网络后重新连接 | Windows：「网络设置 → 管理已知网络 → 忘记」；macOS：「网络 → 高级 → 移除」 |
| 第 4 步 | 检查 Portal 认证状态 | 打开浏览器访问任意 HTTP 网站（如 http://detect.company.com），检查是否被 Portal 拦截 |
| 第 5 步 | 重启无线网卡 | Windows：设备管理器 → 网络适配器 → 禁用/启用无线网卡 |
| 第 6 步 | 重启电脑 | 彻底释放网络协议栈缓存 |

### 常见错误代码

- **「无法连接到该网络」**：账号密码错误或证书过期，请参考本文「连接配置」章节确认凭证
- **「有限的访问权限」**：已连接 WiFi 但未通过 Portal 认证，请打开浏览器完成认证
- **「IP 地址获取失败」**：DHCP 地址池耗尽（高峰时段），等待 5 分钟后重试或联系 IT

### 仍然无法连接？

如执行以上全部步骤后仍无法连接，请联系 IT 服务台（分机 8888），提供以下信息以便快速定位问题：

- 你的位置（楼层+区域，如 3F 西区）
- 设备型号和操作系统版本
- 具体错误信息截图或错误码

## 访客 WiFi 使用说明

1. 访客在 WiFi 列表中选择 **Company-Guest**
2. 输入密码 `Guest@2025`（前台公示，每季度更换）
3. 连接后打开浏览器，自动跳转至 Portal 认证页
4. 输入手机号码，获取并填写短信验证码
5. 认证完成后可访问互联网，有效期 8 小时

---

## 使用规范与注意事项

1. 禁止私自架设无线路由器、随身 WiFi 等设备，私自接入设备会产生 DHCP 污染影响整层网络
2. 禁止使用 WiFi 万能钥匙等密码共享工具连接企业网络
3. 实验室内研发设备请使用有线网络连接，避免无线干扰影响精密仪器
4. 无线网络性能受办公区人数和物理遮挡影响，大文件下载请优先使用有线网络
