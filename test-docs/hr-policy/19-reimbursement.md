# 费用报销管理制度

> 文件编号：FIN-2024-001  
> 版本号：V3.4  
> 生效日期：2024年3月1日  
> 审批人：首席财务官（CFO）、财务总监  
> 适用范围：某科技有限公司全体正式员工

---

## 概述

为规范公司费用管理，保障业务活动的正常开展，加强成本管控，某科技有限公司（以下简称"公司"）财务部制定本**费用报销**管理制度。本制度规定了差旅费、招待费、办公用品费及其他日常费用的报销标准、流程、发票要求及审批权限等事项。

公司员工须遵守"真实、合规、及时、节约"的**报销**原则。一切费用报销须以实际发生的、与本岗位工作直接相关的、合规合法的消费为前提，严禁虚报、多报、重复报销。

---

## 适用范围

本制度适用于公司全体正式员工因公产生的以下费用类型：

- **差旅费**：国内/国际出差产生的交通、住宿、餐补等费用
- **招待费**：因业务需要对外招待客户/合作伙伴产生的餐饮、茶歇费用
- **办公用品费**：部门统一采购的文具、耗材、设备配件等
- **培训/会议费**：参加外部培训、行业会议产生的报名费及差旅费
- **其他业务费用**：经部门负责人和财务部审批同意的其他合理业务支出

实习员工及劳务派遣人员因公产生的费用，经直属领导审批后参照本制度执行。

---

## 报销流程

公司使用OA系统进行**费用报销**全流程在线管理：

```
提交报销单 → 直属领导审批 → 财务审核 → 出纳付款 → 到账
```

### 详细步骤

1. **保管凭证**：消费发生后，收集并妥善保管所有发票、消费小票及相关凭证。电子发票须保存原始PDF或OFD文件。
2. **登录OA**：登录公司OA系统 → 进入「财务中心」→ 点击「费用报销」模块。
3. **选择类型**：根据消费性质选择报销类型——差旅、招待、办公用品、培训/会议、其他。
4. **填写明细**：逐条填写报销明细，包括：日期、事由（简明扼要说明业务目的）、金额、费用类别。差旅报销须同时填写出差申请单编号。
5. **上传发票**：上传发票照片或扫描件。电子发票直接上传原始PDF文件。发票须清晰完整，关键信息（发票号码、日期、金额、销售方名称、税号）可辨识。
6. **提交审批**：确认无误后提交，报销单自动流转至审批流程。

### 审批链路

| 金额区间 | 审批链路 |
|----------|----------|
| 单次报销 ≤ 1,000元 | 直属领导 → 财务审核 |
| 1,000元 < 单次报销 ≤ 10,000元 | 直属领导 → 部门负责人 → 财务审核 |
| 单次报销 > 10,000元 | 直属领导 → 部门负责人 → CFO/财务总监 → 财务审核 |

## 发票要求

### 发票基本信息

- **发票抬头**：某科技有限公司
- **税号**：公司统一社会信用代码（详见OA财务中心公告栏或公司开票信息卡片）
- **发票类型**：增值税普通发票或增值税专用发票均可

### 发票规范

1. 电子发票和纸质发票具有同等法律效力，均可用于报销。
2. 发票须在消费发生后**30天内**提交报销申请，超过30天原则上不予受理（特殊情况附说明经部门负责人审批）。
3. 跨年度发票最迟于次年1月15日前提交，逾期不予受理。
4. 发票内容须与实际消费一致，不得以其他消费类别的发票替代（如以餐饮发票替代办公用品消费）。
5. 单张发票金额超过5,000元的，须附消费明细清单或合同。

## 差旅报销标准

### 交通标准

| 类别 | 标准 | 备注 |
|------|------|------|
| 市内交通 | 实报实销 | 优先使用公共交通工具 |
| 高铁/动车 | 二等座 | 行程超6小时可申请一等座 |
| 飞机 | 经济舱 | 总监及以上可商务舱 |
| 出租车/网约车 | 实报实销 | 往返机场/火车站/客户现场 |

### 住宿标准

| 城市级别 | 标准（每晚） | 备注 |
|----------|-------------|------|
| 一线城市（北上广深） | 500元 | 含税含早 |
| 其他城市 | 350元 | 含税含早 |
| 境外出差 | 按出差地发布标准 | 详见《因公出境管理制度》 |

超出标准部分原则上由个人自理，特殊情况（如举办会议需要入住指定酒店）须提前报批。

### 出差餐补

- **出差餐补标准**：100元/天/人（无需发票，按实际出差天数计算）
- 出差天数按自然日计算（出发日至返回日，含交通日）
- 会议/培训期间如主办方提供餐饮的，当日餐补减半

### 报销款到账时间

审批通过后，报销款通常在**5-7个工作日**内由财务部出纳统一发放至员工工资卡。每月15日、30日为固定付款日（遇节假日顺延至最近工作日），报销单在上述日期前完成全部审批的优先付款。

---

## 注意事项

1. 严禁将个人消费、家庭消费以公务报销名义提交，一经发现按《员工奖惩管理制度》严重违纪处理，追回报销款项并处以2倍罚款。
2. 报销单填写须完整、准确、规范。填写不完整或附件不全的报销单，财务部将退回要求补充，退回次数超过2次的将记入当季度部门费用管理评分。
3. 各部门每月报销总额受部门预算管控，超出月度预算的须由部门负责人提交书面说明并经CFO审批。
4. 电子发票请通过官方渠道下载原始文件，不得修改或PS发票图片，违者按虚假报销处理。
