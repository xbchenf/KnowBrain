# Microsoft 365 企业版安装与激活指南

> 版本：V2.3 | 生效日期：2025-04-01 | 适用范围：某科技有限公司正式员工
> 制定部门：IT 运维部 | 软件支持热线：分机 8002

## 概述

某科技有限公司已采购 Microsoft 365 企业版（E3）批量授权，覆盖全体正式员工。本指南说明 Office 办公套件（Word、Excel、PPT、Outlook、Teams、OneNote）的安装、激活和常见问题处理。

每位员工可使用企业授权在最多 5 台设备上安装 Office，包括公司配发电脑、个人电脑、平板和手机。离职后授权自动失效，无需手动操作。

## 安装步骤

### 前提条件

- Windows 10（版本 22H2 或更高）或 Windows 11
- 至少 4GB 可用磁盘空间
- 管理员权限（公司配发电脑默认已具备）
- 稳定的网络连接

### 安装流程

1. 访问公司软件门户：**https://portal.company.com/software**
2. 在「办公软件」分类中找到「Microsoft 365 企业版」，点击下载安装包。
3. 运行下载的 `OfficeSetup.exe`，安装程序将自动完成下载和安装（约 10-20 分钟，取决于网络速度）。
4. 安装完成后，打开任意 Office 应用（如 Word），使用企业邮箱和密码登录激活。
5. 激活成功后，Excel、PPT、Outlook、Teams、OneNote 等组件均可正常使用。

### Mac 用户

Mac 版安装包同样在公司软件门户可下载。安装后使用企业账号登录激活。Mac 版功能与 Windows 版基本一致，部分高级宏功能可能存在差异。

## 安装失败常见原因

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 安装中途报错 | 之前安装过其他版本 Office（如 Office 2016、Office 2019） | 先使用 Microsoft SARA 工具完全卸载旧版本 |
| 提示磁盘空间不足 | C 盘可用空间 < 4GB | 清理磁盘或更换安装路径 |
| 安装后无法激活 | Windows 系统未更新至最新版本 | 运行 Windows Update 安装所有待处理更新 |
| 登录后提示无授权 | 账户未加入企业许可组 | 联系 IT 服务台（分机 8002）添加授权 |

## 企业授权政策

- **授权范围**：每位正式员工 5 台设备（含 PC、Mac、平板、手机）。实习生和外包人员授权另行申请。
- **授权期限**：在职期间有效，离职后自动失效。
- **使用限制**：禁止将企业授权用于商业用途或非工作相关的项目。禁止将安装包或授权信息分享给非公司人员。
- **续期**：授权每年自动续期，无需员工操作。如收到「授权即将过期」的邮件，请忽略并通过企业微信确认是否为钓鱼邮件。

## 常见问题

**Q: 家里个人电脑可以安装公司的 Office 吗？**  
A: 可以。使用企业账号登录即可激活，占用 5 台设备配额中的 1 台。

**Q: 换新电脑后旧电脑的 Office 怎么处理？**  
A: 在旧电脑上退出企业账号即可释放授权配额，不退出也不影响使用（离职后统一失效）。
