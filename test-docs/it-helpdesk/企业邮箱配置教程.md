# 企业邮箱配置教程

> 邮箱系统：腾讯企业邮箱 | 域名：@knowbrain.com | 更新日期：2026-04-10

---

## 一、邮箱账号说明

每位正式员工入职当天自动创建企业邮箱：

- **邮箱格式**：`姓名拼音@knowbrain.com`（如 `zhangsan@knowbrain.com`）
- **重名处理**：拼音重复时加数字后缀（如 `zhangsan2@knowbrain.com`）
- **初始密码**：`KB2024@姓名首字母大写` 如张三为 `KB2024@ZS`，首次登录后强制修改
- **邮箱容量**：标准 50GB，部门负责人 100GB

> 📌 邮箱账号可在 OA 系统 → 个人信息 → 账号信息 中查看。

## 二、邮箱登录方式

### 2.1 网页版（Webmail）

- 地址：https://mail.knowbrain.com
- 支持扫码登录（企业微信扫码）
- 推荐收藏到浏览器书签

### 2.2 客户端配置

#### Outlook（Windows/macOS）— 推荐

| 步骤 | 操作 |
|------|------|
| 1 | 打开 Outlook → 文件 → 添加账户 |
| 2 | 输入邮箱地址：`zhangsan@knowbrain.com` |
| 3 | 选择「Exchange」或「IMAP」 |
| 4 | 输入密码（非初始密码，需先在网页版修改） |
| 5 | 等待自动配置完成 |

**Outlook 手动配置参数（如自动配置失败）：**

| 参数 | 值 |
|------|-----|
| 接收服务器 (IMAP) | `imap.exmail.qq.com` |
| IMAP 端口 | `993` |
| IMAP 加密 | SSL/TLS |
| 发送服务器 (SMTP) | `smtp.exmail.qq.com` |
| SMTP 端口 | `465` |
| SMTP 加密 | SSL/TLS |
| 用户名 | 完整邮箱地址 |

#### 手机邮件客户端

**iOS 邮件：**
1. 设置 → 邮件 → 账户 → 添加账户 → 其他 → 添加邮件账户
2. 填写姓名、邮箱地址、密码、描述
3. 选择 IMAP，填写收发服务器参数（同上表）
4. 存储后即可使用

**Android（使用 Outlook App）：**
1. Google Play / 应用商店搜索「Microsoft Outlook」下载
2. 打开 App → 添加账户 → 输入邮箱地址
3. 自动识别为 Exchange 账户，输入密码完成

#### Foxmail（Windows）

1. 下载地址：https://www.foxmail.com
2. 安装后打开 → 账号管理 → 新建 → 选择「腾讯企业邮箱」
3. 输入邮箱和密码，自动完成配置

## 五、邮件安全策略

### 5.1 发送限制

| 限制项 | 限制值 |
|--------|--------|
| 单封邮件最大收件人数 | 200 人 |
| 单封附件总大小 | 50 MB |
| 单日发送上限 | 500 封 |
| 单个附件最大 | 30 MB |

> 超出 30MB 的大文件请使用公司文件共享链接（上传至 NAS → 生成分享链接）

### 5.2 外发邮件管控

- 发送到外部邮箱（非 @knowbrain.com）时，邮件标题自动添加 `【外部】` 标记
- 包含敏感关键词（合同、报价、薪资、源代码）的外发邮件将触发**上级审批**流程
- 外发邮件超过 10 个外部收件人需主管审批

### 5.3 钓鱼邮件识别

如收到以下特征的邮件，请**不要点击链接或下载附件**，立即转发至 `it-help@knowbrain.com` 举报：

- 发件人伪装成 CEO/高管但实际邮箱地址不对
- 要求提供密码、验证码、银行账号
- 附件为 `.exe` `.js` `.vbs` `.bat` 等可执行格式
- 内容制造紧迫感："您的邮箱将在 24 小时内停用"
- 链接地址与显示文字不一致（鼠标悬停可查看真实地址）

## 六、常见问题

### Q1：忘记密码 / 账号被锁定

1. 登录网页版 https://mail.knowbrain.com，点击「忘记密码」
2. 通过绑定的手机号或企业微信验证重置
3. 如手机号已变更，联系 IT 运维（分机 8888）后台重置
4. 连续 5 次密码错误账号自动锁定 30 分钟

### Q2：Outlook 提示「无法连接服务器」

1. 检查电脑是否能正常上网
2. 公司内网环境无需 VPN；居家办公需先连接 VPN
3. 检查 Outlook 版本：需 ≥ Outlook 2019 或 Microsoft 365
4. 删除已保存的邮箱账户，重新添加

### Q3：手机收不到新邮件通知

- iOS：设置 → 邮件 → 账户 → 获取新数据 → 选择「推送」或「每 15 分钟」
- Android：Outlook App → 设置 → 通知 → 开启「新邮件通知」
- 确认手机未开启「勿扰模式」或「专注模式」

### Q4：邮件误删如何恢复

1. 网页版登录 → 已删除邮件 → 勾选目标邮件 → 移动 → 收件箱
2. 已删除文件夹中邮件保留 30 天，超期彻底删除
3. 彻底删除的邮件需 IT 管理员后台恢复（7 天内可恢复）

### Q5：收到「邮箱容量已满」警告

1. 清理「已删除邮件」文件夹
2. 清理大附件邮件：网页版搜索 `has:attachment larger:10M`
3. 将历史邮件归档到本地 PST 文件（Outlook）：文件 → 工具 → 清理旧项目
4. 提交工单申请扩容（需部门负责人审批）
