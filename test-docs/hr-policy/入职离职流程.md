# 入职与离职管理流程

> 版本：V3.0 | 生效日期：2026-01-01 | 发布部门：人力资源部

---

## 第一部分：入职流程

### 一、入职前准备（HR 侧）

录用人选接受 Offer 后，HR 启动以下入职准备工作：

| 时间节点 | 事项 | 责任人 |
|---------|------|--------|
| 入职前 2 周 | 发送《录用通知书》（Offer Letter） | 招聘 HR |
| 入职前 2 周 | 发送入职材料清单和体检指引 | 招聘 HR |
| 入职前 1 周 | 创建员工档案、生成员工号 | HR 运营 |
| 入职前 1 周 | 提交 IT 工单：创建域账号/邮箱/各系统账号 | HR 运营 |
| 入职前 1 周 | 安排工位、准备办公设备和入职礼包 | 行政 + IT |
| 入职前 1 周 | 邮件通知部门负责人和导师 | 招聘 HR |
| 入职前 3 天 | 确认所有账号已开通，设备已就位 | IT + HR |
| 入职前 1 天 | 发送《入职报到提醒》邮件（含时间、地点、携带材料） | HR 运营 |

### 二、入职当天流程

| 时间 | 事项 | 责任人 | 地点 |
|------|------|--------|------|
| 09:00-09:15 | 前台签到、领取访客卡 | 前台 | 1F 前台 |
| 09:15-09:45 | HR 办理入职手续 | HR | HR 会议室 |
| 09:45-10:15 | IT 发放设备、配置账号 | IT | 员工工位 |
| 10:15-11:00 | 信息安全培训（必修） | 信息安全官 | 培训室 |
| 11:00-11:30 | 参观办公环境、介绍各功能区 | 行政 | 全楼层 |
| 11:30-12:00 | 导师带领认识团队成员 | 导师 | 工位区 |
| 12:00-13:30 | 欢迎午餐（部门安排） | 部门 | — |
| 14:00-17:00 | 岗位培训、熟悉业务 | 导师 | 工位 |

### 三、入职手续所需材料

| 序号 | 材料 | 用途 | 要求 |
|------|------|------|------|
| 1 | 身份证原件 | 身份核验 | 有效期内 |
| 2 | 学历学位证书原件 | 学历核实 | 最高学历 |
| 3 | 离职证明原件 | 劳动关系核实 | 上一家单位开具 |
| 4 | 招商银行储蓄卡 | 工资发放 | 本人实名卡 |
| 5 | 一寸白底照片 × 2 | 工牌制作 + 档案 | 近 6 个月内 |
| 6 | 近 3 个月体检报告 | 健康档案 | 二甲以上医院 |
| 7 | 社保公积金转出单 | 社保接续 | 如有 |

### 四、劳动合同签订

| 合同要素 | 说明 |
|---------|------|
| 合同期限 | 首次签订 3 年（含试用期 3 个月） |
| 续签 | 第二次签订 5 年，第三次签订无固定期限 |
| 签订时间 | 入职当天签订，一式两份（公司 + 员工各一份） |
| 签署方式 | 电子签章（上上签平台），入职当天 HR 指引完成 |

> 📌 试用期 3 个月的约定符合《劳动合同法》对 3 年合同期的规定。

### 五、试用期管理

| 阶段 | 时间 | 事项 |
|------|------|------|
| 入职第一周 | 第 1-5 天 | 新员工培训 + 了解业务 + 设定试用期 OKR |
| 月度回顾 | 每月末 | 导师 1-on-1，跟踪 OKR 进展和融入情况 |
| 转正考核启动 | 入职满 2.5 个月 | 员工自评 + 导师评价 + 部门负责人评价 |
| 转正结果通知 | 入职满 3 个月 | 通过 → 正式员工；延期 → 延长试用期 1-2 个月；未通过 → 终止合同 |

**转正考核内容：**

| 维度 | 权重 | 评估方式 |
|------|------|---------|
| 试用期 OKR 达成 | 50% | 结果评分 |
| 岗位技能匹配度 | 30% | 上级评估 |
| 文化与团队融入 | 20% | 导师 + 同事反馈 |

## 第二部分：离职流程

### 六、辞职申请

| 人员类型 | 提前通知期 | 提交方式 |
|---------|-----------|---------|
| 正式员工 | 30 天 | OA → HR 服务 → 离职申请 |
| 试用期员工 | 3 天 | OA → HR 服务 → 离职申请 |

> 📌 提交离职申请前，建议先与直属上级面谈沟通。

### 七、离职交接（标准周期：2-4 周）

#### 7.1 工作交接

| 事项 | 说明 |
|------|------|
| 工作清单 | 列出进行中和已完成的项目/任务 |
| 文档移交 | 技术文档、项目文档、知识库更新至 Confluence/Wiki |
| 代码交接 | Git 仓库权限转移、代码 Review 遗留分支处理 |
| 客户/供应商交接 | 当前对接客户联系人、进行中商务事项移交 |
| 权限回收 | 各系统管理员权限移交给指定同事 |
| 交接确认 | 接收人签字确认，部门负责人审核 |

#### 7.2 资产归还

| 资产 | 归还对象 | 备注 |
|------|---------|------|
| 笔记本电脑 + 充电器 | IT | 系统重置后签收 |
| 显示器/键盘/鼠标 | IT | — |
| 工牌 | 行政 | — |
| 门禁卡 | 行政 | — |
| U 盾/加密狗 | IT | — |
| 公司手机/SIM 卡 | IT | 如有 |
| 图书 | 行政 | 从公司图书馆借阅的 |

#### 7.3 财务结算

| 结算项目 | 说明 |
|---------|------|
| 截止离职日薪资 | 按实际工作日计算 |
| 剩余年假折算 | 按规定公式折算成薪资发放 |
| 剩余调休折算 | 按日薪折算发放 |
| 未报销费用 | 需在离职前提交完毕 |
| 培训违约金 | 如在培训服务期内离职（分期递减） |
| 竞业限制补偿 | 如适用，离职后按月支付 |

### 八、离职面谈

HR 在员工离职前安排一对一离职面谈（约 30 分钟），了解：

- 离职原因（职业发展 / 薪酬 / 管理 / 文化 / 个人原因）
- 对公司管理和文化的真实反馈
- 是否有挽回可能
- 离职后竞业限制是否启动

面谈内容脱敏后纳入《离职分析月报》，不包含个人身份信息。

### 九、离职手续办理

| 节点 | 时限 | 事项 |
|------|------|------|
| 离职当天 | 18:00 前 | IT 回收所有系统权限 |
| 离职当天 | 18:00 前 | HR 开具《离职证明》 |
| 离职后 15 天内 | — | 社保减员、公积金减员/封存 |
| 次月 10 日 | — | 最后一次薪资发放 |

### 十、离职后义务

| 义务 | 说明 |
|------|------|
| 保密义务 | 在职期间掌握的商业秘密和技术机密永久保密 |
| 竞业限制 | 如适用，最长 2 年，公司每月支付补偿金（离职前月薪的 30%） |
| 不挖角义务 | 离职后 1 年内不得招揽、挖角公司员工 |
| 知识产权归属 | 在职期间产生的知识产权仍归属公司 |

### 十一、特殊情形

| 情形 | 处理 |
|------|------|
| 退休 | 达到法定退休年龄，HR 协助办理退休手续 |
| 合同到期不续签 | 公司提前 30 天通知，支付 N 个月经济补偿 |
| 协商解除 | 双方协商一致，签订《协商解除劳动合同协议书》 |
| 公司单方解除 | 需符合《劳动合同法》第 39/40 条规定 |
