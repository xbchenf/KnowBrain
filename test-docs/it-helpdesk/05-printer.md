# 办公打印机安装、使用与维护规范

> 文档版本：v2.2 | 生效日期：2025-02-15 | 责任部门：IT 基础设施部 / 行政部 | 适用范围：某科技有限公司总部各楼层办公区

---

## 概述

某科技有限公司在总部各楼层部署了多台企业级多功能打印设备，支持打印、复印、扫描和传真功能。所有打印机通过有线网络接入公司内网，员工使用企业账号认证后方可使用，打印记录记入部门配额系统。

打印机根据楼层区域分布，员工可就近使用任意一台设备。彩色打印和高速打印为受限功能，普通员工默认为黑白打印，如需使用受限功能须向部门助理提交申请。

打印耗材（纸张、墨粉、硒鼓）由行政部统一采购和更换，员工发现耗材不足时请通过企业微信「行政报修」入口通知行政部，请勿自行更换以免损坏设备。

## 打印机安装步骤

### 自动安装（推荐）

1. 确认目标打印机已开机（触摸屏亮起为正常状态）并已连接网络（屏幕右上角显示网络图标）
2. 在电脑上打开文件资源管理器，地址栏输入 `\\print-server` 访问公司打印服务器
3. 打印服务器页面会自动搜索你所在子网内的所有可用打印机，展示列表包含：打印机型号、所在位置、功能类型
4. 在列表中找到目标打印机（建议根据楼层和区域选择最近的一台），双击打印机名称
5. 系统自动下载并安装对应驱动，安装过程约 1~2 分钟
6. 安装完成后，在「Windows 设置 → 设备 → 打印机和扫描仪」中可看到已添加的打印机
7. 打开任意文档选择打印，在打印机下拉列表中选择刚添加的打印机，打印测试页确认配置正确

### 手动安装（自动安装失败时）

1. 下载对应型号的驱动安装包（驱动下载地址：`\\print-server\drivers`，请根据打印机型号选择对应文件夹）
2. 运行驱动安装程序，按向导完成驱动安装
3. Windows 设置 → 设备 → 打印机和扫描仪 → 添加打印机或扫描仪
4. 选择「我需要的打印机不在列表中」→「按名称选择共享打印机」
5. 输入打印机网络路径，格式为 `\\print-server\<打印机名称>`（打印机名称可在设备机身上的标签找到）
6. 点击「下一步」完成添加

### Mac 系统安装

1. 系统偏好设置 → 打印机与扫描仪 → 点击「+」添加
2. 在「IP」标签页中输入打印机 IP 地址（可在打印机触摸屏的「设备信息」页面查看），协议选择「AirPrint」或「IPP」
3. 系统自动识别打印机型号并下载驱动

## 扫描功能使用

以 Canon iR-ADV C5535（3F 西区）为例：

1. 将文档正面朝上放入输稿器，或正面朝下放在扫描玻璃板上
2. 在触摸屏上点击「扫描并发送」
3. 选择目标方式：
   - **扫描至邮箱**：输入你的企业邮箱地址，扫描件以 PDF 附件发送
   - **扫描至 USB**：将 U 盘插入打印机侧面 USB 接口，扫描文件直接存入 U 盘
   - **扫描至共享文件夹**：选择预先配置的部门共享文件夹
4. 设置扫描参数：色彩模式（彩色/黑白）、分辨率（默认 300dpi，文字文档推荐 200dpi）、文件格式（PDF/JPEG/TIFF）
5. 按「开始」键执行扫描

## 常见问题处理

### 打印机离线或无法打印

1. 检查打印机是否开机、触摸屏是否有报错信息（如卡纸、缺墨、缺纸）
2. 在电脑上打开「控制面板 → 设备和打印机」，右键点击打印机 →「查看正在打印的内容」
3. 如果打印队列中有卡住的任务，右键 →「取消所有文档」清空队列
4. 右键打印机 →「打印机属性」→「端口」→ 确认端口配置正确
5. 重启 Print Spooler 服务：运行 `services.msc` → 找到「Print Spooler」→ 右键「重新启动」

### 打印出来是乱码

通常由驱动版本不匹配导致。请卸载当前打印机驱动（在设备管理器中右键打印机 → 删除设备），然后回到上文的「打印机安装步骤」从打印服务器重新自动安装。

### 卡纸处理

1. 不要强行拉扯卡住的纸张，以免残留纸屑损坏加热组件
2. 按照打印机屏幕上显示的卡纸位置动画指引，依次打开对应盖板
3. 顺着走纸方向平稳拉出卡纸
4. 如无法自行处理，联系行政部或 IT 服务台

## 注意事项

1. 打印含敏感信息的文档时，请使用打印机的「安全打印」功能——在电脑上勾选「安全打印」后设置取件密码，到打印机旁刷卡或输入密码后文档才会被打印输出
2. 彩色打印成本为黑白打印的 5 倍，非必要请使用黑白打印
3. 打印完文档请及时取走，以免泄漏信息或丢失
4. 废弃打印件请使用碎纸机销毁，注意纸张分类回收
