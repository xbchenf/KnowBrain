# 企业账号密码重置与安全管理规定

> 文档版本：v3.1 | 生效日期：2025-03-01 | 责任部门：信息安全部 | 适用范围：某科技有限公司全体员工

---

## 概述

企业账号是员工访问公司各类信息系统的唯一身份凭证，包括但不限于：企业邮箱、VPN、OA 系统、代码仓库、打印系统。为确保账号安全，所有密码须遵循复杂度要求并定期更换。本文档规定了忘记密码时的自助重置流程、密码策略要求以及异常情况的人工处理方式。

某科技有限公司采用统一身份认证平台（IDaaS），一个密码通行全部内部系统。若密码泄漏或遗忘，员工可通过自助平台或联系 IT 服务台完成密码重置。

---

## 自助密码重置流程

### 前提条件

使用自助重置前，请确认已提前在「某科安全中心」App 中绑定以下**至少一项**备用验证方式：

- 个人手机号（推荐，支持短信验证码）
- 个人备用邮箱（非企业邮箱）

未提前绑定备用验证方式的员工，自助重置功能不可用，须走人工重置通道（见下文「人工重置」章节）。

### 操作步骤

1. 在浏览器中访问自助密码重置平台：**https://pwd.company.com**
2. 在账号输入框中输入你的企业账号（格式为员工编号，如 `ZH00123`，或邮箱前缀，如 `zhangsan`）
3. 点击「下一步」，系统展示你已绑定的验证方式列表
4. 选择一种验证方式：
   - **手机短信验证**：系统向你绑定的手机号发送 6 位验证码，在页面中填入验证码
   - **备用邮箱验证**：系统向你绑定的个人邮箱发送验证链接，点击链接完成验证
5. 验证通过后进入「设置新密码」页面，输入新密码并确认
6. 重置成功后，系统强制注销该账号在所有设备上的已有登录会话，须用新密码重新登录

### 密码复杂度要求

| 要求项 | 说明 |
|--------|------|
| 最小长度 | 8 位 |
| 字符组合 | 必须同时包含大写字母、小写字母、数字和特殊字符（如 !@#$%） |
| 禁止内容 | 不得包含账号名、员工姓名拼音、生日、手机号等易猜测信息 |
| 历史限制 | 不能与最近 5 次使用过的密码相同 |
| 有效期 | 每 90 天强制更换一次，到期前 7 天每日弹窗提醒 |

## 人工重置通道

以下情况须通过 IT 服务台人工处理：

- 未提前绑定手机号或备用邮箱，自助重置不可用
- 绑定的手机号或邮箱已停用，无法接收验证信息
- 自助重置平台出现系统故障

人工重置流程：

1. 联系 IT 服务台（分机 **8888**，工作时间 9:00-18:00；紧急情况拨打值班手机 138xxxx1234）
2. 提供员工编号和身份证后 6 位进行身份核验
3. IT 工程师核实身份后发送临时密码至你绑定的手机号或备用邮箱
4. 使用临时密码登录后，系统强制跳转至修改密码页面，须立即设置新密码
5. 临时密码有效期为 30 分钟，过期未使用须重新申请

## 注意事项与安全提醒

1. **不要在浏览器中保存密码**：公共或共享电脑上禁止勾选「记住密码」
2. **不要使用统一密码**：公司账号密码不要与个人社交账号、网购账号等密码相同
3. **警惕钓鱼邮件**：IT 部门不会通过邮件或即时通讯索要你的密码，收到可疑邮件请转发至 security@company.com
4. **密码泄露应急处理**：如怀疑密码已泄露，立即登录 pwd.company.com 自助修改密码，并通知信息安全部

## 修改密码（已知旧密码）

已知当前密码且密码未过期的正常修改场景，可登录任意企业内部系统，在「个人设置 → 修改密码」页面完成修改，无需走重置流程。
