# 考勤与假期管理制度

> 版本：V4.1 | 生效日期：2026-01-01 | 发布部门：人力资源部

---

## 一、总则

本制度规范 KnowBrain 全体员工的工作时间、考勤管理、加班管理及各类假期的申请与审批流程。旨在保障正常办公秩序，兼顾员工工作生活平衡。

---

## 二、工作时间

### 2.1 标准工时制

| 项目 | 内容 |
|------|------|
| 工作日 | 周一至周五（法定节假日除外） |
| 工作时间 | 9:00 — 18:00 |
| 午休时间 | 12:00 — 13:00（1 小时） |
| 周工作时长 | 40 小时 |

### 2.2 弹性工作制

公司实行弹性工作制，具体要求：

- **核心工作时间**：10:00 — 16:00（所有员工必须在线/在岗）
- **最早到岗**：7:00
- **最晚离岗**：22:00
- 满足核心工作时间 + 日满 8 小时即可，不强制 9:00 打卡

弹性工作制不适用于以下岗位：
- 前台接待、展厅讲解
- 客户支持（轮班制）
- IT 运维值班

### 2.3 特殊工时制

| 岗位类型 | 工时制度 | 说明 |
|---------|---------|------|
| 高管 | 不定时工作制 | 不参与日常考勤 |
| 销售 | 不定时工作制 | 以外勤/客户拜访记录为准 |
| 运维/客服 | 综合计算工时制 | 按月累计工时 |

## 三、考勤管理

### 3.1 打卡方式

| 方式 | 操作 |
|------|------|
| 企业微信打卡 | 进入办公区域后自动触发打卡（基于蓝牙 Beacon） |
| 手动打卡 | 企业微信 → 工作台 → 考勤打卡 |
| 外勤打卡 | 外出拜访/出差时使用，自动记录 GPS 位置 |

### 3.2 打卡时间要求

- **上午上班**：不晚于 10:00（弹性窗口内均可）
- **下午下班**：不早于 16:00（需满足 8 小时工作时长）
- 非弹性岗位必须执行 9:00 前打卡

### 3.3 迟到与早退

| 情形 | 处理 |
|------|------|
| 单次迟到 ≤ 30 分钟 | 不扣薪，但计入月度统计 |
| 月累计迟到 > 3 次 | 超出部分按事假折算（每 3 次 = 0.5 天事假） |
| 单次迟到 > 30 分钟 | 超出时间按事假扣除 |
| 早退（早于 16:00） | 按实际早退时长折算事假 |
| 忘记打卡 | 每月 ≤ 3 次可 OA 补卡（主管审批），超过按迟到处理 |

### 3.4 缺勤

| 类型 | 说明 |
|------|------|
| 旷工半天 | 未请假、未打卡、无合理理由，扣 1 天薪资 + 书面警告 |
| 旷工一天 | 扣 2 天薪资 + 书面警告 |
| 连续旷工 3 天 | 按自动离职处理 |
| 月累计旷工 5 天 | 按自动离职处理 |

## 四、加班管理

### 4.1 加班定义

| 加班类型 | 定义 | 补偿方式 |
|---------|------|---------|
| 工作日加班 | 超过 18:00 继续工作且≥ 2 小时 | 1.5 倍加班费 或 1:1 调休 |
| 休息日加班 | 周六/周日加班 | 2 倍加班费 或 1:1 调休 |
| 法定节假日加班 | 元旦/春节/清明/五一/端午/中秋/国庆 | 3 倍加班费（不可调休） |

### 4.2 加班申请流程

1. **事前申请**：OA → 考勤管理 → 加班申请，填写加班事由和预计时长
2. **主管审批**：需在加班发生前审批（紧急情况可事后 24 小时内补批）
3. **加班确认**：加班结束时在企业微信「加班确认」中打卡
4. 未经审批的自行加班不计入加班考勤

### 4.3 调休规则

| 规则 | 说明 |
|------|------|
| 调休计算 | 加班 4 小时 = 0.5 天调休，8 小时 = 1 天调休 |
| 调休有效期 | 加班日起 90 天内使用，过期作废 |
| 调休使用 | 优先使用即将到期的调休 |
| 离职调休 | 剩余调休按日薪折算随最后薪资发放 |

### 4.4 加班限制

| 限制 | 上限 |
|------|------|
| 日加班上限 | 3 小时（特殊情况不超过 4 小时） |
| 月加班上限 | 36 小时 |

## 五、假期制度

### 5.1 法定年假

| 累计社会工龄 | 年假天数 | 备注 |
|------------|---------|------|
| 1 年 ≤ 工龄 < 10 年 | 5 天 | — |
| 10 年 ≤ 工龄 < 20 年 | 10 天 | — |
| 工龄 ≥ 20 年 | 15 天 | — |

**年假使用规则：**

- 新入职员工：当年年假 = 法定天数 × (剩余月数/12)，向上取整至 0.5 天
- 离职员工：当年年假 = 法定天数 × (已工作月数/12)，多休部分按日薪扣回
- 年假最小请假单位：0.5 天
- 年假可以分段使用，不可跨年度结转（次年 3 月 31 日前需用完上年年假）
- 建议提前 3 个工作日申请

### 5.2 带薪病假

| 条件 | 待遇 |
|------|------|
| 病假 1-2 天 | 无需医院证明，薪资 100% |
| 病假 3-30 天 | 需提供二级甲等以上医院诊断证明，薪资 80% |
| 病假 31-90 天 | 薪资 60%（按当地最低工资标准保底） |
| 病假 > 90 天 | 进入医疗期，按《劳动合同法》相关规定执行 |

### 5.3 婚假

- **天数**：10 天（含工作日，不含法定节假日）
- **条件**：领取结婚证后 1 年内一次性连续使用
- **证明材料**：结婚证原件（HR 核验后返还）
- **薪资**：全额带薪

### 5.4 产假 / 陪产假 / 哺乳假

| 假期 | 天数 | 说明 |
|------|------|------|
| 产假 | 158 天 | 含产前 15 天，难产加 15 天，多胞胎每多一胎加 15 天 |
| 产前检查假 | 按需 | 正常出勤对待，需提供产检证明 |
| 流产假 | 15-42 天 | 根据怀孕时长和医嘱 |
| 陪产假 | 15 天 | 配偶预产期前后使用 |
| 哺乳假 | 1 小时/天 | 子女满 1 周岁前（可分 2 次，每次 30 分钟） |

> 📌 产假期间薪资由生育保险基金支付生育津贴，公司补足差额至正常薪资。

### 5.5 事假

- **性质**：无薪假
- **上限**：年度累计不超过 20 个工作日
- **薪资扣除**：日薪 = 月基本工资 / 21.75 天
- **审批**：1-2 天由主管审批，3 天以上需部门负责人审批

### 5.6 其他假期

| 假期 | 天数 | 条件 |
|------|------|------|
| 丧假 | 3 天 | 直系亲属（父母、配偶、子女、公婆、岳父母） |
| 育儿假 | 5 天/年 | 子女 0-3 周岁，父母各享 |
| 护理假 | 10 天/年 | 独生子女，父母年满 60 周岁住院 |
| 考试假 | 1 天/次 | 经批准的职业资格考试 |
| 搬家假 | 1 天 | 同城搬家，每年 1 次 |

### 5.7 请假流程总览

```
OA 系统 → 考勤管理 → 请假申请
   ↓
填写：假期类型 / 起止时间 / 请假原因 / 工作交接人
   ↓
审批链：
  ├─ 年假/调休（1-2 天）：主管
  ├─ 年假/调休（3+ 天）：主管 → 部门负责人
  ├─ 病假（1-2 天）：主管
  ├─ 病假（3+ 天）：主管 → HR
  ├─ 婚假/产假/陪产假：主管 → HR
  └─ 事假：同层级 + 1 天以上加部门负责人
   ↓
审批通过 → 系统自动同步到考勤记录
```

## 六、假期冲突处理

| 情形 | 处理方式 |
|------|---------|
| 年假与法定节假日重叠 | 法定节假日不计入年假天数 |
| 产假与年假重叠 | 产假顺延（年假可待产假结束后使用） |
| 病假与年假重叠 | 可选择按病假或年假处理（员工自主选择） |
| 多人同时请假 | 同部门同一时段请假人数不超过部门总人数的 30% |
