# 软件安装与采购申请管理制度

> 版本：V3.1 | 生效日期：2025-05-01 | 适用范围：某科技有限公司全体员工
> 制定部门：IT 运维部 & 采购部 | 软件申请支持：分机 8005

## 概述

为规范某科技有限公司软件资产的采购、安装和使用，保障信息安全并控制 IT 成本，所有工作用软件的安装须遵守本制度。公司实行「白名单自助安装 + 非白名单审批」的双轨管理模式。

员工安装任何软件前，应先确认该软件是否已在公司软件白名单中，以选择正确的获取方式。

## 软件白名单（自助安装）

公司维护一份经过安全审查的「软件白名单」，白名单内的软件员工可自行下载和安装，无需额外审批。最新白名单请访问：**https://portal.company.com/software**

白名单当前涵盖以下类别：

| 类别 | 示例 | 获取方式 |
|------|------|---------|
| 办公软件 | Microsoft 365、WPS Office（仅阅读） | 公司软件门户 |
| 开发工具 | VS Code、JetBrains 全家桶（社区版）、Git、Docker Desktop | 公司软件门户 或 官网下载 |
| 设计工具 | Figma（Web 版）、Adobe Acrobat Reader | 官网下载 |
| 通讯协作 | 企业微信、飞书、钉钉、Zoom | 公司软件门户 |
| 系统工具 | 7-Zip、Notepad++、Everything | 公司软件门户 |

白名单每月更新一次，由 IT 安全团队评估新申请后纳入。员工可在自助平台提交「建议加入白名单」的软件推荐。

## 非白名单软件申请流程

若所需软件不在白名单中，须走完整审批流程：

1. **提交申请**：IT 自助平台 → 软件申请 → 填写以下信息：
   - 软件名称和版本
   - 用途说明（具体到项目或工作任务）
   - 预计费用（如有采购成本）
   - 使用人数
2. **直属领导审批**：确认软件确为工作需要。
3. **IT 安全评估**（2-3 个工作日）：检查软件许可协议、安全漏洞记录、数据隐私合规性。
4. **采购 / 部署**：免费软件由 IT 统一部署；付费软件进入采购流程（金额 < 5000 元部门审批即可，>= 5000 元需总监级审批）。
5. **安装验收**：IT 在指定设备上安装并验证。

## 安全红线

以下行为严格禁止：

- **安装盗版或未经授权的软件**：违反软件版权法，公司和个人均可能承担法律责任。使用盗版软件一经发现，立即通报直属领导和 HR。
- **安装破解版软件**：破解工具常携带恶意代码，是病毒和勒索软件的主要感染源之一。
- **使用个人账号注册云服务**：涉及公司数据的云服务（如笔记工具、项目管理工具）须通过公司统一采购的企业版，确保数据归属和合规。

## 开发工具与开源软件

开发工具和开源软件（如 VS Code 插件、npm 包、Python 库）虽通常免费，但仍需通过安全审查。重点审查项：

- 开源许可证是否与公司产品兼容（GPL 系列许可证有潜在法律风险）
- 是否有已知安全漏洞（CVE 数据库比对）
- 维护状态（是否会成为「废弃软件」带来技术债务）

## 涉及数据的云服务额外审批

若软件涉及将公司数据传输至第三方服务器（如在线文档协作工具、AI 助手插件、云存储同步盘），除上述审批外，还需通过**数据合规审批**（数据安全部负责，3-5 个工作日），确保符合《数据安全法》和公司数据分类分级制度。

## 处理周期汇总

| 场景 | 周期 |
|------|------|
| 白名单软件自助安装 | 即时 |
| 免费软件申请（已在白名单推荐列表中） | 2-3 个工作日 |
| 通用商业软件 | 5-7 个工作日 |
| 专业软件（需采购） | 视采购流程，通常 2-4 周 |
| 涉及数据合规的云服务 | 额外 3-5 个工作日 |
