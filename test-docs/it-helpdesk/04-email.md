# 企业邮箱配置与使用指南

> 文档版本：v3.0 | 生效日期：2025-01-01 | 责任部门：IT 基础设施部 | 适用范围：某科技有限公司全体员工

---

## 概述

某科技有限公司使用 Microsoft Exchange Server 2019 作为企业邮箱后端，为全体员工提供统一的企业邮箱服务。邮箱格式为 `姓名拼音@company.com`（如 `zhangsan@company.com`），邮箱容量默认 50GB，支持发送最大 50MB 附件。

企业邮箱可在多种客户端上配置使用：Outlook 桌面版（推荐）、Foxmail、Mac 邮件、iOS/Android 原生邮件客户端、以及网页版（OWA）。本文档涵盖各主流客户端的配置参数和操作步骤。

新员工入职时企业邮箱由系统自动创建，入职当天可正常使用。离职员工邮箱在 HR 确认离职手续完毕后 30 天自动归档封存。

---

## 邮箱配置参数总览

无论使用哪种邮件客户端，核心配置参数均相同：

| 参数 | 配置值 | 说明 |
|------|--------|------|
| 邮件地址 | 姓名拼音@company.com | 如 zhangsan@company.com |
| 账号 | 员工编号@company.com | 如 ZH00123@company.com |
| 接收协议 | Exchange（推荐）或 IMAP | Exchange 支持日历/联系人/任务同步 |
| IMAP 服务器 | imap.company.com | 端口 993，SSL 加密 |
| SMTP 服务器 | smtp.company.com | 端口 587，TLS 加密 |
| 网页版地址 | https://mail.company.com | 浏览器直接访问，无需配置 |

> **推荐**：优先使用 Exchange 模式配置，可同步邮件、日历、联系人和任务。IMAP 仅同步邮件，适合对日历同步无需求的用户。

## Outlook 桌面版配置

### 首次配置（Exchange 自动发现）

1. 打开 Outlook，如为首次启动会显示「欢迎使用 Microsoft Outlook」向导
2. 在邮箱地址栏中输入你的企业邮箱地址（如 `zhangsan@company.com`）
3. 点击「连接」，Outlook 自动通过 Autodiscover 发现 Exchange 服务器配置
4. 在弹出的登录窗口中输入企业账号和密码
5. 配置完成后 Outlook 开始同步邮件数据，首次同步视数据量大小可能需要 5~15 分钟

### 手动配置（IMAP 模式）

如果自动发现失败，可手动配置 IMAP：

1. Outlook → 文件 → 添加账户 → 选择「手动设置或其他服务器类型」
2. 选择「POP 或 IMAP」，点击「下一步」
3. 填写以下信息：
   - 你的姓名：中文姓名
   - 电子邮件地址：zhangsan@company.com
   - 账户类型：IMAP
   - 接收邮件服务器：**imap.company.com**，端口 **993**，加密方法 **SSL/TLS**
   - 发送邮件服务器：**smtp.company.com**，端口 **587**，加密方法 **STARTTLS**
4. 点击「其他设置」→「发送服务器」→ 勾选「我的发送服务器要求验证」
5. 点击「下一步」→「完成」

## Foxmail 配置

1. 打开 Foxmail，点击右上角菜单 →「账号管理」→「新建」
2. 输入企业邮箱地址和密码，点击「创建」
3. Foxmail 自动尝试查找服务器配置，如自动配置失败请手动填写：
   - 接收服务器类型选择 **IMAP**
   - IMAP 服务器：**imap.company.com**，端口 **993**，SSL 开启
   - SMTP 服务器：**smtp.company.com**，端口 **587**，SSL 开启
4. 点击「确定」完成配置

## 手机端配置（iOS/Android）

### iOS 原生邮件

1. 打开「设置」→「邮件」→「账户」→「添加账户」
2. 选择「**Microsoft Exchange**」（推荐）或「其他」→「添加邮件账户」
3. 输入企业邮箱地址和密码，iOS 自动尝试 Exchange 发现
4. 如选择 Exchange 模式，勾选需要同步的项目：邮件、通讯录、日历、提醒事项
5. 如使用 IMAP 手动配置，服务器地址与上文一致

### Android Outlook App（推荐）

1. 在应用商店搜索并下载「Microsoft Outlook」App
2. 打开 App，输入企业邮箱地址，点击「继续」
3. 选择「Exchange」账户类型，输入密码
4. App 自动完成配置并开始同步

## 网页版（OWA）访问

不需要任何客户端配置，使用浏览器访问 **https://mail.company.com**，输入企业账号和密码即可使用。网页版功能与 Outlook 桌面版基本一致，支持邮件收发、日历管理、联系人管理、任务分配。

推荐以下场景使用网页版：

- 公共电脑或他人电脑上临时查看邮件（使用隐私模式/无痕窗口）
- 客户端故障时的应急访问
- 出差时使用酒店商务电脑

## 常见问题

| 问题 | 解决办法 |
|------|---------|
| Outlook 配置时提示「无法建立加密连接」 | 检查电脑系统时间是否正确，时间偏差超过 5 分钟会导致 SSL 证书验证失败 |
| 手机邮件 App 频繁弹出密码提示 | 账号密码可能已过期，请修改密码后重新输入，或在 App 中删除账户后重新添加 |
| 发出的邮件被退回 | 检查收件人地址是否正确，检查邮件大小是否超过 50MB 附件限制 |
| 网页版登录提示「该账号已被锁定」 | 连续输错密码 5 次触发账号锁定，等待 15 分钟后重试或联系 IT 服务台 |
| 需要找回已删除邮件 | 删除的邮件在「已删除邮件」文件夹中保留 30 天，可自行恢复。超过 30 天请联系 IT 从备份中恢复 |

## 邮箱使用规范

1. 禁止使用企业邮箱注册个人社交媒体、网购平台等非工作用途
2. 涉及公司机密的邮件请使用加密发送功能
3. 定期清理邮箱，避免邮箱满导致无法接收新邮件（邮箱容量使用率达到 90% 时系统会发送提醒）
4. 不要打开来源不明的附件和链接，收到可疑邮件请转发至 security@company.com
