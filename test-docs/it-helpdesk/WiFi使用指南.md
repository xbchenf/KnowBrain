# 公司办公网络 WiFi 使用指南

> 更新日期：2026-05-20 | 维护部门：IT 运维部

---

## 一、WiFi 网络列表

公司各办公区提供以下 WiFi 网络：

| SSID 名称 | 频段 | 用途 | 适用人群 |
|-----------|------|------|---------|
| `KB-Office` | 5GHz | 日常办公主网络 | 正式员工 |
| `KB-Office-2.4G` | 2.4GHz | 兼容老旧设备 | 正式员工 |
| `KB-Guest` | 2.4G/5GHz | 访客临时网络 | 访客、面试者 |
| `KB-IoT` | 2.4GHz | 打印机、监控等设备 | IT 管理 |

> ⚠️ **不要连接 `KB-IoT`**：该网络仅限物联网设备使用，手动连接会导致 MAC 地址被拉黑。

## 二、连接方式

### 2.1 KB-Office（正式员工）

**方法一：域账号自动连接（推荐）**

- **Windows 域控电脑**：首次开机后 WiFi 自动配置，无需手动输入密码
- 如未自动连接，点击任务栏 WiFi 图标 → 选择 `KB-Office` → 使用域账号登录

**方法二：预共享密钥（PSK）**

1. 打开 WiFi 设置，选择 `KB-Office`
2. 输入密码：`KB@Office2024!Secure`（每季度通过企业微信统一通知变更）
3. 连接成功后，浏览器会弹出 Portal 认证页面
4. 输入域账号和密码完成认证

**方法三：802.1X 企业认证（BYOD 设备）**

个人手机或平板连接 `KB-Office` 时：

| 参数 | 值 |
|------|-----|
| EAP 方法 | PEAP |
| 阶段 2 认证 | MSCHAPv2 |
| 身份 | 域账号（knowbrain\员工号） |
| 密码 | 域密码 |
| CA 证书 | 不验证（首次连接时选择"不验证"） |

### 2.2 KB-Guest（访客网络）

1. 连接 `KB-Guest`（无需密码）
2. 浏览器自动跳转到访客认证页面
3. 输入手机号获取短信验证码
4. 输入验证码后即可上网，有效期 **8 小时**
5. 访客网络只能访问互联网，**不能访问公司内网系统**

> 📌 前台接待人员可为访客生成一次性使用码，有效期 24 小时。

## 五、常见问题排查

### Q1：找不到 KB-Office 信号

1. 确认所在位置是否为办公区域（卫生间、楼梯间信号较弱）
2. Windows：打开「设备管理器」→ 网络适配器 → 确认无线网卡驱动正常，无黄色感叹号
3. 检查是否误触了笔记本电脑的物理 WiFi 开关（部分 ThinkPad 机型按 Fn+F8 开启）
4. 重启无线网卡：设置 → 网络和 Internet → 高级网络设置 → 禁用 → 重新启用

### Q2：连接后提示「无 Internet 访问」

- **Portal 认证过期**：打开浏览器访问任意网页，会重定向到认证页面重新登录
- **IP 地址冲突**：命令行执行 `ipconfig /release && ipconfig /renew` 重新获取 IP
- **MAC 地址被限制**：如多次认证失败 IP 会被临时封禁 30 分钟，联系 IT 解封

### Q3：WiFi 信号满格但网速很慢

1. 检查是否误连了 `KB-Office-2.4G`（2.4GHz 频段干扰大，速度慢）
2. 切换至 5GHz 频段的 `KB-Office`
3. 会议室人多时 WiFi 会降速，大型会议建议使用有线网络
4. 关闭大流量应用（视频会议、云盘同步、系统更新）再测试

### Q4：macOS 连接 KB-Office 提示证书错误

1. 删除已保存的 KB-Office 网络配置：系统偏好设置 → 网络 → Wi-Fi → 高级 → 选中 KB-Office → 点击「-」删除
2. 重新连接 KB-Office
3. 弹出证书提示时选择「继续」或「信任」

### Q5：手机无法保持 WiFi 连接

- Android：设置 → WLAN → 高级 → 在休眠状态下保持 WLAN 连接 → 选择「始终」
- iOS：设置 → 无线局域网 → 关闭「私有无线局域网地址」（针对 KB-Office）

## 六、有线网络

每个工位配有 **2 个 RJ45 千兆网口**（桌面线槽内），推荐以下场景使用有线：

- 视频会议（腾讯会议/Zoom > 10 人）
- 大文件上传下载（> 500MB）
- 远程桌面 / SSH 连接（需要稳定低延迟）

使用方式：网线插入桌面网口即可，DHCP 自动获取 IP，无需额外配置。如需固定 IP 请提交工单申请。
