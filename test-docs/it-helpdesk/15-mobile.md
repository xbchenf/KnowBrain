# 移动办公设备配置与安全管理制度

> 版本：V2.0 | 生效日期：2025-03-01 | 适用范围：某科技有限公司全体员工
> 制定部门：IT 运维部 & 信息安全部 | 移动办公支持：分机 8006

## 概述

为支持员工在移动场景下高效办公，同时保障企业数据安全，本制度规定了公司邮箱在手机端的配置方法、企业 IM 工具（企业微信 / 飞书 / 钉钉）的安装使用、以及移动设备的安全管理要求。

所有接入公司数据（邮件、文件、IM 消息）的移动设备必须遵守本制度中的安全要求。未满足安全基线（如无锁屏密码、未安装 MDM 证书）的设备将被拒绝访问公司资源。

## 企业邮箱手机端配置

### iPhone / iPad

1. 打开「设置」→「邮件」→「账户」→「添加账户」。
2. 账户类型选择「**Microsoft Exchange**」。
3. 输入完整企业邮箱地址（`name@company.com`）和密码。
4. 系统自动发现 Exchange 服务器配置。如提示服务器地址，输入 `mail.company.com`。
5. 选择需要同步的内容：邮件（必选）、日历、通讯录、提醒事项（推荐全选）。
6. 点击「存储」完成配置。

### Android 手机

1. 在应用商店搜索并下载 **Microsoft Outlook** App（推荐使用 Outlook 原生 App，而非手机自带邮件客户端）。
2. 打开 Outlook App → 添加账户 → 选择「Exchange」。
3. 输入企业邮箱和密码 → 允许 Outlook 管理设备权限（这是 MDM 安全策略的一部分）。
4. 等待邮件、日历和联系人同步完成。

### 常见问题

- **配置失败提示「无法验证账户」**：确认密码是否正确；确认账户未被锁定（连续输错 5 次密码将锁定 15 分钟）。
- **邮件不同步**：检查手机网络连接；确认 Outlook App 有后台刷新权限。

## 企业 IM 安装

公司支持以下企业 IM 工具（根据部门使用的平台选择安装）：

| 平台 | 安装方式 | 适用群体 |
|------|---------|---------|
| **企业微信** | 应用商店搜索「企业微信」→ 使用企业账号登录 | 全公司通用 |
| **飞书** | 应用商店搜索「飞书」→ 使用企业账号登录 | 产研团队 |
| **钉钉** | 应用商店搜索「钉钉」→ 使用企业账号登录 | 运营 / 商务团队 |

登录方式为 SSO 统一认证，使用企业邮箱和密码即可。

## 移动设备安全要求（强制执行）

- **锁屏密码**：必须设置至少 6 位数字密码或生物识别（指纹/面部识别）。禁止使用无密码滑动解锁。
- **MDM 设备管理证书**：配置企业邮箱或登录企业 IM 时，系统会提示安装 MDM（Mobile Device Management）管理证书。此证书允许 IT 在公司数据泄露时远程擦除公司数据，不会访问个人数据（照片、短信、通话记录等）。
- **沙箱隔离**：公司数据（邮件、文件、IM 消息）与个人数据通过 Microsoft Intune 沙箱机制隔离，个人应用无法访问公司数据，反之亦然。
- **系统更新**：手机操作系统须保持最新版本（最多落后一个大版本），旧版本可能存在已知安全漏洞。

## 手机丢失应急处理

如手机丢失或被盗，须**立即**执行以下步骤：

1. 第一时间联系 IT 服务台（分机 8006）或在 IT 自助平台使用「远程擦除」功能。
2. IT 将远程擦除该设备上的**所有公司数据**（邮件、IM 消息、缓存的附件）。个人数据不受影响。
3. 修改企业账号密码，防止密码在设备解锁后被利用。
4. 在新设备上重新配置后可恢复正常办公。

## 适用范围与例外

- **适用设备**：接入公司数据的员工个人手机和平板。
- **不适用**：公司未配发手机，员工使用个人设备办公属自愿行为。如因安全原因不愿在个人设备上安装 MDM 证书，可向 IT 申请公司配发的办公手机（需部门总监审批）。
- **国际差旅**：部分国家/地区可能限制企业 VPN 或 MDM 功能，出行前请咨询 IT。
