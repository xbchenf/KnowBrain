# 会议室设备使用与技术指南

> 文档版本：v3.2 | 生效日期：2025-03-10 | 责任部门：IT 基础设施部 / 行政部 | 适用范围：某科技有限公司总部全部会议室（含大型培训室）

---

## 概述

某科技有限公司在总部各楼层设置了多间智能化会议室，涵盖 4 人洽谈间、8-12 人标准会议室以及可容纳 60 人的大型培训室。所有会议室均配备以下标准化设备：

- 壁挂式液晶电视或投影仪（带电动幕布）
- 无线投屏接收器（支持 AirPlay、Miracast、Google Cast）
- 会议室专用 PC（已预装 Teams、Zoom、WPS Office）
- 全向麦克风 + 摄像头套件（视频会议专用）
- HDMI 有线投屏备用线缆

会议室通过 Outlook 日历系统统一预订，员工可在 Outlook 中查看会议室忙闲状态并预约。本文档聚焦于会议室设备的操作使用，预订规则请参阅行政部发布的《会议室使用管理制度》。

## 无线投屏操作指南

无线投屏是最便捷的投屏方式，无需任何线缆即可将笔记本屏幕内容投射到会议室大屏上。

### 通用步骤

1. 进入会议室后，按电视/投影仪遥控器上的电源键开机
2. 设备启动后，电视画面会显示**投屏指引画面**，包含以下关键信息：
   - 会议室设备名称（如 `MeetingRoom-301`）
   - 当次连接的 4 位 **PIN 码**（每次开机随机生成，保障安全）
   - WiFi 网络名称确认（确保连接的是 Company-WiFi）
3. 确保你的笔记本已连接公司 WiFi（**Company-WiFi**），且与会议室设备在同一网络
4. 在笔记本上发起投屏连接（按操作系统选择下方对应方式）

### Windows 投屏

按下键盘快捷键 **Win + K**，系统弹出投屏面板，在设备列表中找到会议室设备名称（如 `MeetingRoom-301`），点击连接，在弹出的 PIN 码输入框中输入电视屏幕上显示的 4 位 PIN 码，等待 3~5 秒完成连接。

### macOS 投屏

点击菜单栏「控制中心」图标 →「屏幕镜像」，在设备列表中选择会议室设备名称，输入电视屏幕上显示的 PIN 码。

### 手机/平板投屏

- **iPhone/iPad**：控制中心 → 屏幕镜像，选择会议室设备，输入 PIN 码
- **Android**：设置 → 连接 → 投屏（或 Smart View 等厂商自定义名称），选择会议室设备，输入 PIN 码

> 提示：如投屏面板中未出现会议室设备，请确认笔记本已连接 Company-WiFi（而非 Company-Guest 或手机热点），忘记 WiFi 后重新连接再试。

## 视频会议操作指南

会议室专用 PC 已预装 Microsoft Teams 和 Zoom 客户端，摄像头和全向麦克风通过 USB 连接至会议 PC，开机即用。

### 使用 Teams 开会

1. 打开会议室 PC 登录系统（会议室 PC 使用会议室专用公用账号自动登录）
2. 确认摄像头和麦克风已连接 — 观察摄像头镜头旁的蓝色指示灯是否亮起，检查 PC USB 接口处全向麦克风的连接线是否插紧
3. 点击桌面上的「Microsoft Teams」图标启动客户端，使用你的企业账号登录
4. 在 Teams 日历中找到预定的会议，点击「加入」按钮进入
5. 进入会议后，点击「打开视频」和「取消静音」启用摄像头和麦克风
6. 会议开始前建议在「设备设置」中确认音视频设备选择正确：
   - 扬声器：选择「会议室全向麦」（Echo Cancelling Speakerphone）
   - 麦克风：选择「会议室全向麦」
   - 摄像头：选择「Logitech MeetUp」

### 使用 Zoom 开会

操作流程与 Teams 类似，在会议室 PC 桌面上点击「Zoom」图标启动，登录你的 Zoom 企业账号，在「会议」标签页中找到预定的会议并加入。首次使用 Zoom 时须在「设置 → 音频」中将扬声器和麦克风切换为会议室全向麦克风设备。

### 常见问题

| 问题 | 解决办法 |
|------|---------|
| 摄像头无画面 | 检查 USB 连接线是否松动，尝试重新插拔；在 Teams/Zoom 设备设置中确认选中了正确的摄像头设备 |
| 对方听不到声音 | 检查全向麦克风是否被静音（麦克风上物理静音键指示灯红色表示已静音）；检查 Teams/Zoom 中麦克风设备选择是否正确 |
| 声音有回声/啸叫 | 将全向麦克风放在会议室中央，避免靠近音箱或墙壁；降低扬声器音量；将笔记本自身的麦克风和扬声器关闭（避免与会议室设备冲突） |
| 无法共享屏幕 | 在 Teams/Zoom 会议中点击「共享」按钮，选择要共享的窗口或整个屏幕 |

## 有线投屏（备用方案）

当无线投屏不畅或 Wi-Fi 信号不稳定时，可使用有线 HDMI 方式作为备用方案：

1. 找到会议桌上标注「HDMI」的线缆（通常收纳在桌面线槽内）
2. 将 HDMI 线缆一端连接至笔记本的 HDMI 接口（若笔记本无 HDMI 接口，请自备 USB-C 转 HDMI 适配器，或向 IT 服务台借用）
3. 使用电视/投影仪遥控器，按「信号源」或「Source」键，在信号源列表中选择对应的 HDMI 端口（如 HDMI 1、HDMI 2）
4. 笔记本画面即显示在大屏上

## 使用规范与礼仪

1. 会议结束后请关闭电视/投影仪电源，断开投屏连接（Win+K → 断开），退出个人 Teams/Zoom 账号
2. 使用完会议室后，将遥控器、转接头等物品归位至桌面收纳盒
3. 请勿在会议 PC 上安装未经 IT 部门批准的软件
4. 会议中如需讨论机密信息，请在 Teams 中选择「仅限特定人员」的会议选项

如会议前发现设备故障无法使用，请立即拨打 IT 服务台紧急热线 **分机 8899**（工作日 8:00-20:00），IT 工程师将在 15 分钟内到场排查。如会议室无线网络异常，IT 现场提供便携式 4G 热点作为临时应急方案。
