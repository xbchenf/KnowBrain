---

## 二、安装打印机驱动

### 2.1 Windows（域控电脑自动安装）

公司域控电脑首次开机时自动推送打印机：

1. 点击「开始」→「设置」→「蓝牙和其他设备」→「打印机和扫描仪」
2. 应能看到以 `KB-` 开头的打印机列表
3. 如未显示，重启电脑等待组策略生效

### 2.2 Windows 手动添加（非域控电脑）

1. 下载驱动：
   - HP M528：`\\nas.knowbrain.local\Software\Drivers\HP_M528_Win11.exe`
   - Canon C3530：`\\nas.knowbrain.local\Software\Drivers\Canon_C3530_Win.exe`
2. 安装驱动后，添加打印机 → 选择「按名称选择共享打印机」
3. 输入：`\\printserver.knowbrain.local\KB-HP-2F`（替换为目标打印机名称）

### 2.3 macOS

1. 系统偏好设置 → 打印机与扫描仪 → 点击「+」
2. 选择「IP」标签页
3. 输入打印机 IP 地址（见第一节设备名称旁标签）
4. 协议选择「IPP（Internet Printing Protocol）」
5. 驱动自动从 Apple 更新服务器下载

## 三、打印操作

### 3.1 普通打印

1. 在任意应用中 Ctrl+P（macOS: Cmd+P）打开打印对话框
2. 选择目标打印机
3. 设置参数：

| 参数 | 推荐值 |
|------|--------|
| 纸张大小 | A4 |
| 双面打印 | 是（默认，可改为单面） |
| 色彩 | 黑白（默认，彩色需手动选择） |
| 每面页数 | 1 页（PPT 可设 2 页/面） |
| 份数 | 1 份（多份请走复印机） |

### 3.2 安全打印（私密文档）

1. 打印对话框 → 打印机属性 → 作业存储 → 选择「安全打印」
2. 设置 4-8 位数字取件码（PIN）
3. 发送打印任务后，到打印机触摸屏操作：
   - 点击「作业存储」→「安全打印作业」
   - 选择你的作业 → 输入 PIN 码 → 打印

### 3.3 扫描到邮箱

1. 将原稿放入打印机输稿器（朝上）或玻璃面板（朝下）
2. 打印机触摸屏点击「扫描」→「扫描到电子邮件」
3. 输入你的企业邮箱地址（首次使用需手动输入，后续自动记住）
4. 选择扫描参数：
   - 文件格式：PDF（多页）/ JPEG（单页图片）
   - 分辨率：300 dpi（文档）/ 600 dpi（图片）
   - 色彩：黑白（文档）/ 彩色（含图片）
   - 双面扫描：原稿为双面时开启
5. 点击「发送」，扫描件将发送到你的邮箱

## 四、耗材与报修

### 4.1 更换墨盒/硒鼓

当打印机面板提示「碳粉不足」或打印变淡时：

1. 到各楼层茶水间旁耗材柜自取耗材
2. HP M528 使用 CF289A 硒鼓
3. Canon C3530 墨粉型号见打印机面板提示
4. 按照打印机面板动画指引更换（约 2 分钟）
5. 旧耗材放入柜内的回收箱（公司统一环保处理）

### 4.2 卡纸处理

1. **不要用力拉扯卡住的纸张**
2. 按照打印机屏幕动画指引，打开对应舱门
3. 顺着纸张走纸方向轻轻取出
4. 如纸张碎裂，确保取出所有碎片
5. 关闭舱门，打印机会自动恢复

### 4.3 报修流程

卡纸无法自行处理 / 打印质量异常 / 设备报错代码 → 

1. 记下打印机型号和错误代码（如 HP 报错 `13.02.00`）
2. OA 提交工单 → IT 服务 → 打印机故障
3. 或直接联系 IT 分机 8888
4. 打印机上张贴了 IT 资产标签（含设备编号），报修时提供此编号

## 五、使用规范

| 规范 | 说明 |
|------|------|
| 环保打印 | 默认双面黑白打印，非必要不使用彩色和单面 |
| 及时取件 | 打印后 5 分钟内取走文档，防止信息泄露 |
| 信息安全 | 含敏感信息的打印件使用「安全打印」功能 |
| 禁止事项 | 禁止打印私人资料、禁止用彩色打印机打印装饰品 |
| 下班检查 | 最后离开者确认打印机无遗留文档 |

## 七、常见问题

### Q1：发送打印任务后打印机没反应

1. 确认打印机是否处于「就绪」状态（非休眠/离线）
2. 检查是否选错了打印机（常见：选了隔壁楼层的同名打印机）
3. Windows：「设置 → 打印机 → 打开打印队列」查看是否有卡住的任务，取消后重新发送
4. 重启 Print Spooler 服务：Win+R → 输入 `services.msc` → 找到 Print Spooler → 右键重新启动

### Q2：打印出来的文档是乱码

1. 确认文档语言设置正确（中文文档用 GB2312/UTF-8）
2. PDF 文件 → 打印对话框 → 高级 → 勾选「作为图像打印」
3. 更新打印机驱动

### Q3：macOS 找不到打印机

1. 确认 Mac 已连接公司内网（有线或 KB-Office WiFi）
2. 系统偏好设置 → 打印机与扫描仪 → 右键列表空白处 → 重置打印系统
3. 重新添加打印机（见第二节）
